Para resolver este problema le pediremos al usuario que introduzca por teclado 3 numeros
una vez introducidos comprobaremos si los 3 numeros son iguales, si es el caso escribiremos que son 3 numeros iguales
en caso contrario seguiremos con el programa

haremos todas las comprobaciones posibles de la siguiente forma:

comprobaremos si el primer numero introducido es igual o mayor que el segundo

 si esta regla se cumple le preguntaremos si el tercero es mayor o igual que el primero, si es el caso quiere decir que el orden es:
 tercer numero > primer numero > segundo numero
 si esta regla no se cumple le preguntaremos si el segundo es igual o mayor que el tercero y que en ese caso ya sabremos que el primer numero es el mayor
y solo nos quedara saber el orden de los otros dos

si no se cumple que el primero sea igual o mayor que el segundo comprobaremos si el primero es mayor o igual que el tercero

ya que si se cumple esa regla sabremos que el orden será: segundo numero > primer numero > tercer numero
y si no se cumple ya sabremos que el primer numero será el mas pequeño de todos
por lo que solo quedará preguntar si el tercero es mayor o igual que el segundo para decidir el orden en el que se escribirán los otros dos
